## How to Use

### Start Everything
```bash
# Local Docker
docker compose up -d

# MCP Server
python mcp_server.py server
# OR end-to-end workflow
python mcp_server.py workflow

# Kubernetes (optional)
kubectl apply -f k8s-deployment.yaml
kubectl apply -f k8s-monitoring.yaml
```

### Make API Calls
```bash
# Check health
curl http://localhost:8888/health

# List services
curl http://localhost:8888/services

# Call any service
curl -X POST http://localhost:8888/call \
  -H "Content-Type: application/json" \
  -d '{
    "service": "codex_engine",
    "method": "get_status",
    "params": {}
  }'
```

### Run Kaggle Pipeline
```bash
python kaggle_destroyer_submission.py
```
