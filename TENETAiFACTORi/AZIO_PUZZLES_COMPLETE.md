## Puzzle Types Explained

### 1. **Extraction Puzzle**
Extract the underlying pattern from a wish.

```
Given: "Build system that collapses to 1"
Find:
  • The attractor (what everything collapses to)
  • The invariant (what doesn't change)
  • The generalization (infinite applications)
  
Reward: 100 pts
```

### 2. **Optimization Puzzle**
Optimize the solution under constraints.

```
Constraints:
  • Minimize time complexity
  • Minimize space complexity
  • Maximize accuracy
  • Maximize generalizability

Find the optimal approach.
Reward: 150 pts
```

### 3. **Pattern Puzzle**
Find ALL patterns in the wish.

```
Find:
  1. The attractor
  2. The divergences
  3. The symmetries
  4. The recursive structure
  5. The generalizations

Reward: 200 pts
```

### 4. **Game Theory Puzzle**
Compete against the system.

```
Rules:
  • You vs System
  • System doubles puzzles each round
  • First to solve all variants wins
  
Can you win against exponential growth?
Strategy: Extract patterns faster than system generates.
Reward: 300 pts
```

### 5. **Meta-Puzzle** (Highest Value)
A puzzle ABOUT puzzles.

```
Questions:
  1. How many puzzles from this wish?
  2. At what iteration → infinite?
  3. What's the puzzle that solves all?
  4. Can a puzzle solve itself?
  5. Create a cycle (puzzle generates itself)

Reward: 500 pts
```

### 6. **Constraint Satisfaction**
Satisfy hard and soft constraints.

```
Hard constraints (MUST satisfy):
  • Generalizable to ≥5 domains
  • Learnable (teachable)
  • Provably optimal
  • Terminates in finite time

Soft constraints (SHOULD satisfy):
  • Elegant (simple, beautiful)
  • Scalable
  • Verifiable

Reward: 250 pts
```

### 7. **Symbolic Logic Puzzle**
Formalize in first-order logic.

```
Define:
  • Predicates
  • Axioms
  • Transformation rules
  • Goal state

Prove:
  1. Solution exists (∃ solution)
  2. Uniqueness (≤ 1 solution)
  3. Correctness (⊢ correct)
  4. Minimality (minimal proof)

Reward: 400 pts
```
